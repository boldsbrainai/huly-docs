O módulo de gestão de testes da Huly permite que você crie, organize e execute testes de forma eficiente dentro do seu espaço de trabalho. Com este módulo, você pode rastrear os status dos testes, visualizar os resultados, vincular os testes a problemas no Tracker e gerar tickets de trabalho para os responsáveis.

## Configurando testes

Clique no botão `+` na barra lateral esquerda para iniciar um novo projeto de teste. Por enquanto, os projetos de teste estão isolados dentro do módulo de gestão de testes; no entanto, existem planos para conectar a gestão de testes com outros projetos do espaço de trabalho no futuro próximo.

### Criando uma suíte de testes
Abra o menu suspenso no canto superior esquerdo e selecione `Criar suíte de testes`. Forneça um nome e uma descrição opcional para sua suíte de testes.

### Criando um caso de teste
Dentro de sua suíte de testes, adicione casos de teste clicando em `+ Novo caso de teste`. Insira um nome e uma descrição opcional para o seu caso de teste. Selecione um status (por exemplo, "Pronto", "Em andamento"), e anexe quaisquer arquivos ou imagens relevantes. Em seguida, clique em `Criar`.

### Executando testes

Use as caixas de seleção ao lado de cada caso de teste para escolher quais incluir na execução de testes. Em seguida, clique em `Executar`.

Forneça um título e uma descrição para a sua execução de testes. Sua execução de testes pode incluir detalhes como a configuração do ambiente, instruções de execução e resultados esperados.
