## API da Huly 

A Huly tem uma API básica para você começar suas próprias integrações e extensões. Confira a [documentação da API](https://github.com/hcengineering/huly-examples/tree/main/platform-api#readme) para ver as etapas de configuração e exemplos de uso.

## Ferramenta de importação da Huly

Oferecemos uma ferramenta de importação para ajudar você a importar dados diretamente para o seu workspace da Huly.

Usando nosso Formato Unificado de Importação, você pode migrar seus dados para a Huly convertendo-os em uma estrutura intermediária, legível por humanos. Confira a [documentação da Ferramenta de Importação da Huly](https://github.com/hcengineering/platform/tree/develop/dev/import-tool#readme) para ver as etapas de configuração e exemplos.

Também oferecemos suporte a importações de [Notion](https://github.com/hcengineering/platform/blob/develop/dev/import-tool/docs/notion/README.md) e [ClickUp](https://github.com/hcengineering/platform/blob/develop/dev/import-tool/docs/clickup/README.md).
